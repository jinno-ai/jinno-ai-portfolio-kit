## 🆘 困ったら?

### エラーが出た場合:

**「GITHUB_TOKEN not found」**
→ Step 2をもう一度実行してください

**「Authentication failed」**
→ トークンの権限を確認してください（repo, workflowが必要）

**「Repository already exists」**
→ 正常です! 「y」で更新、「n」でスキップ

**その他のエラー**
→ [AUTOMATION_GUIDE.md](AUTOMATION_GUIDE.md) のトラブルシューティングを参照
