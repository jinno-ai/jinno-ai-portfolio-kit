## ✅ Step 1: Profile README作成 (10分)

### 1-1. 特別なリポジトリを作成

1. [GitHub](https://github.com) にログイン
2. 右上の「+」→「New repository」をクリック
3. 以下を入力:
   ```
   Repository name: jinno-ai  (あなたのユーザー名と同じ!)
   Description: My GitHub Profile
   ☑ Public
   ☑ Add a README file
   ```
4. 「Create repository」をクリック

### 1-2. README.mdを編集

1. 作成されたリポジトリで「README.md」をクリック
2. 鉛筆アイコン(編集)をクリック
3. **すべてのテキストを削除**
4. 提供された `jinno-ai-profile-README.md` の内容を**全てコピー&ペースト**
5. 下の「Commit changes」をクリック
6. 「Commit changes」をもう一度クリック

### 1-3. 確認
